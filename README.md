# Password Strength Checker
A beginner-friendly Python project that checks password strength using multiple security rules.
## Features
- Checks password length
- Detects uppercase letters
- Detects numbers
- Detects special characters
- Gives password strength rating
- Suggests stronger passwords
## Technologies Used
- Python
## Run the Project
```bash
python checker.py
```
## Example Output
```text
Final Strength: STRONG
```
